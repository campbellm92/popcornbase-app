# PopcornBase

PopcornBase: a movie app that gives recommendations based on specified minimum and maximum ratings values.

## Tech

Backend: **Node.JS** and **Express.js**.

Frontend: **Vanilla JS**, **Bootstrap and CSS**, **Handlebars**

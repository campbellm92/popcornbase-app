# PopcornBase

Questa è popcorn base, un'app che offre raccomandazioni di film random in base a un rating minimo e massimo specificato.

## Tech

Backend: **Node.JS** e **Express.js**.

Frontend: **Vanilla JS**, **Bootstrap e CSS**, **Handlebars**
